# Projeto de Disciplina - Banco de Dados

## Sistema Escolhido
Discord - uma plataforma de comunicação que permite aos usuários interagir usando texto, voz e vídeo. A plataforma permite aos usuários participar ou criar suas próprias comunidades, organizadas em torno de vários tópicos, interesses ou grupos de jogo.

## Descriçao
Este projeto tem como objetivo modelar um banco de dados baseado no sistema do Discord. O foco principal foi implementar boa parte das funcionalidades essenciais do sistema utilizando os conceitos aprendidos durante a disciplina de Banco de Dados I. A maior motivaçao do projeto é ver como o Discord funciona "por debaixo do capô". 

## Como rodar
Copiar os codigos em "migrations" e rodar em ordem numerica em algum ambiente como o DB Fiddle, alterando a opçao de Database para PostgreSQL na versao mais recente.
